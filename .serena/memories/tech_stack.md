# Nori Tech Stack

## Frontend
- **React 18** - UI framework
- **TypeScript 5.3** - Type safety
- **Vite 5** - Build tool and dev server (hot reload)
- **Chakra UI 2** - Component library
- **Zustand 5** - State management
- **Emotion** - CSS-in-JS
- **Framer Motion** - Animations
- **CodeMirror 6** - Code editor (for knowledge editing)
- **React Markdown** - Markdown rendering

## Backend
- **Express 4** - HTTP server
- **WebSocket (ws)** - Real-time AI streaming
- **better-sqlite3** - SQLite database
- **get-port** - Automatic port allocation (3000-3009)

## Desktop
- **Electron 28** - Desktop app framework
- **Context isolation enabled** - Security best practice

## AI Integration
- **@ai-sdk/anthropic** - Anthropic Claude SDK
- **ai (Vercel AI SDK)** - AI streaming utilities
- **claude-sonnet-4** - Current model

## Development Tools
- **TypeScript** - Strict mode enabled
- **ESLint** - Linting (TypeScript ESLint)
- **Vitest** - Unit testing
- **Playwright** - E2E testing
- **concurrently** - Run multiple dev processes
- **electron-rebuild** - Rebuild native modules

## Build & Package
- **electron-builder** - Package desktop app (Windows, Mac, Linux)
- **TypeScript Compiler** - Main process build
- **Vite** - Renderer process build
