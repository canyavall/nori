# Excalidraw Diagrams

## How to Use

1. Go to https://excalidraw.com
2. Click "Open" (top left)
3. Select "Load from JSON"
4. Upload any `.json` file from this folder
5. Edit, customize, export as needed

## Available Diagrams

### diagram1-api-request-structure.json
**Shows:** How commands/agents are translated to Anthropic API requests
**Key insight:** System prompt (cached) vs Messages (paid every time)

### diagram2-knowledge-problem.json
**Shows:** The 3 options for loading 150+ knowledge files
**Key insight:** Dynamic loading is the only viable solution

### diagram3-cost-comparison.json
**Shows:** Cost breakdown of Multi-Agent vs Single Session vs Dynamic Loading
**Key insight:** Dynamic loading saves 45% cost with higher quality

## Exporting

In Excalidraw, you can export to:
- PNG (for presentations)
- SVG (for high-quality docs)
- PDF (for reports)
- Share link (for collaboration)
